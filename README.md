# Usage

```bash
tinyserver
```

Available Options

| Option    | Flag              | Description              | Defaults |
| :-------- | :---------------- | :----------------------- | :------- |
| bind ip   | `-i, --bind_ip`   | bind ip                  | `""`     |
| bind port | `-p, --bind_port` | bind port (default 9876) | 9876     |
| whitelist | `-w, --whitelist` | whitelist                | nil      |
